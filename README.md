# Sudoku solver

## Basics

Call sudoku.solve(), pass it a string consisting of nine lines
with nine digits each line. Unknown cells can be any non-numeric character, e.g.

    python sudoku.py <<EOF
    24...8...
    ....147..
    91..73..2
    16...75..
    .2.145.8.
    ..43...71
    8..72..13
    ..183....
    ...4...67
    EOF
